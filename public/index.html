ATA Addon Template Generator

Generates an addon template based on your trigger and directory standards.

ATA standard

Generate template

Requirements

  • Geometries: geometry.rc_ADDONACRONYM.model_name
  • Animations: animation.rc_ADDONACRONYM.animation_name
  • Animation Controllers: controller.animation.rc_ADDONACRONYM.controller_animation_name
  • Render Controllers: controller.render.rc_ADDONACRONYM.render_name
  • Sounds: always prefix ids with rc_ADDONACRONYM
  • Textures in item_texture/terrain_texture: rc_ADDONACRONYM:texture_name
  • Folders with rc_ADDONACRONYM/ (and rc_ADDONACRONYM/common/) are mandatory: all content must live inside them
  • Triggers: use only main.js. Do not create triggers in other files
  • You may add triggers + custom components in main.js, but you must redirect to CREATOR_triggers
  • runInterval delay: implement inside your own trigger using currentTick % (delayInTicks) == 0
