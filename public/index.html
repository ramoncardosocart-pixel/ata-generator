ATA

ATA Addon Template Generator

Gera um addon template baseado no seu padrão de triggers e diretórios.

tema: preto + laranja

Gerar template

Obs: o gerador cria UUIDs novos para os dois manifests e ajusta as dependências BP↔RP automaticamente.

Obrigatoriedades

  • Geometrias: geometry.rc_ADDONACRONYM.model_name
  • Animations: animation.rc_ADDONACRONYM.animation_name
  • Animation Controllers: controller.animation.rc_ADDONACRONYM.controller_animation_name
  • Render Controllers: controller.render.rc_ADDONACRONYM.render_name
  • Sons: sempre prefixo rc_ADDONACRONYM no id
  • Textures no item_texture/terrain_texture: rc_ADDONACRONYM:texture_name
  • Pastas com rc_ADDONACRONYM/ (e rc_ADDONACRONYM/common/) são obrigatórias: todo conteúdo vai dentro delas
  • Triggers: usar somente o main.js. Não criar outros triggers em outros arquivos
  • Pode adicionar triggers + custom components no main.js, mas deve redirecionar para o CREATOR_triggers
  • Delay no runInterval: faça no seu próprio trigger usando currentTick % (delayInTicks) == 0
